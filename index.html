Preventing CSRF

Unveiling Vulnerabilities of Web Attacks Considering Man in the Middle Attack and Session Hijacking

In this article, we propose a methodology to leverage Machine Learning (ML) for the detection of web application vulnerabilities. Web applications are particularly challenging to analyse, due to their diversity and the widespread adoption of custom programming practices. ML is thus very helpful for web application security.
